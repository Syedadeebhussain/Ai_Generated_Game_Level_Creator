🚀 Overview

AutoLevel AI is an AI-powered procedural game level generator that automatically creates game levels based on player performance. Instead of manually designing levels, the system dynamically generates layouts including platforms, obstacles, enemies, and rewards.

The system also supports adaptive difficulty scaling, meaning the next level is generated based on how the user plays. If the player performs well, the next level becomes harder. If the player struggles, the system generates an easier level.

This enables infinite dynamic gameplay and reduces manual level design effort.
